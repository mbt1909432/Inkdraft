# Editor - Markdown 编辑器

![](../screenshot-editor.png)

编辑器是 Inkdraft 的核心功能，支持双模式编辑、实时大纲、智能格式化等特性。

## 双模式编辑

### 富文本模式（默认）
- 所见即所得的编辑体验
- 支持常见的 Markdown 格式（加粗、斜体、标题、列表等）
- 工具栏快速格式化按钮

### 源码模式
- 点击工具栏的 `</>` 按钮切换
- 直接编辑 Markdown 源码
- CodeMirror 驱动，支持语法高亮
- 选中文本时半透明背景，文字清晰可见

## 实时大纲

编辑器右侧显示文档结构大纲：
- 自动提取 h1-h6 标题
- 点击标题快速跳转
- 拖拽调整面板宽度

## 选中文本工具栏

选中任意文本时，自动弹出浮动工具栏：

### 格式化按钮
| 按钮 | 功能 | 快捷键 |
|------|------|--------|
| **B** | 加粗 | Ctrl+B |
| *I* | 斜体 | Ctrl+I |
| U | 下划线 | Ctrl+U |
| ~~S~~ | 删除线 | - |
| `</>` | 行内代码 | Ctrl+` |

### AI 美化功能
点击展开菜单，选择：
- **润色** - 改善写作风格和清晰度
- **扩写** - 添加更多细节和阐述
- **缩写** - 总结并缩短内容
- **修正语法** - 纠正语法和拼写错误
- **自定义** - 输入你的指令

## 代码块

支持多种编程语言的语法高亮：

```python
import pandas as pd

# 数据分析示例
data = pd.read_csv('sales.csv')
summary = data.groupby('region').sum()
print(summary)
```

支持的语言：JavaScript、Python、TypeScript、Go、Rust、Java、C/C++ 等。

## 自动保存

- 每 30 秒自动保存到云端
- 手动保存：Ctrl+S 或点击保存按钮
- 保存状态实时显示：
  - ☁️ 云朵图标 = 已同步
  - ⏳ 旋转图标 = 保存中
  - ☁️‍🚫 带斜杠云朵 = 离线

## 同步状态

工具栏显示云同步状态：
- **已同步** - 文档已保存到云端，显示普通云朵图标
- **同步中** - 正在上传，显示旋转动画
- **离线** - 网络断开，显示带斜杠云朵图标
