# Mobile Experience

![](../screenshot-mobile.png)

Inkdraft is deeply optimized for mobile, making all features accessible on small screens.

## Mobile Layout

### Top Toolbar
- **Back Button** - Return to document list
- **Document Title** - Shows current document name
- **Sync Status** - Cloud sync icon
- **More Menu** - Export, save, settings, etc.

### Bottom Navigation Bar
Four quick action buttons:

| Button | Function |
|--------|----------|
| 📁 | Document list (sidebar) |
| ✏️ | Draft (AI generate) |
| 🤖 | AI Assistant |
| 📋 | Outline view |

## Core Features

### Document Editing
- Touch-optimized editor
- Pinch to zoom text size
- Long press to select text
- Selection toolbar for formatting

### AI Draft Generation
Mobile-specific draft entry:
1. Tap bottom **Draft** button
2. Enter document title and instructions
3. AI generates complete draft
4. Preview, then accept or discard

### AI Chat
- Tap bottom **AI** button to open chat
- Full-screen chat interface
- All desktop AI features available
- Swipe to close

### Outline Navigation
- Tap bottom **Outline** button
- Bottom sheet with document outline
- Tap heading to jump to section

## Export Functions

In top **More Menu**:
- **Copy Markdown** - Copy to clipboard
- **Download .md** - Download Markdown file
- **Export Word** - Generate .docx file
- **Export PDF** - Generate PDF file

## Gestures

| Gesture | Action |
|---------|--------|
| Single tap | Position cursor |
| Long press | Select text |
| Pinch | Zoom page |
| Swipe left | Go back |

## Save Status

Mobile save status appears above bottom nav:
- **"Saving..."** - Syncing
- **"Saved"** - Sync complete (auto-hides after 2s)
- **"Save failed"** - Network error (auto-hides after 3s)

## Offline Usage

- Documents are cached locally
- Offline shows cloud with slash icon
- Auto-sync when network returns
