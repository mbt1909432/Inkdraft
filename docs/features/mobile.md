# Mobile Experience - 移动端体验

![](../screenshot-mobile.png)

Inkdraft 针对移动端进行了深度优化，在小屏幕上也能流畅使用所有功能。

## 移动端布局

### 顶部工具栏
- **返回按钮** - 返回文档列表
- **文档标题** - 显示当前文档名
- **同步状态** - 云同步图标
- **更多菜单** - 导出、保存、设置等

### 底部导航栏
四个快捷操作按钮：

| 按钮 | 功能 |
|------|------|
| 📁 | 文档列表（侧边栏） |
| ✏️ | 起稿（AI 生成） |
| 🤖 | AI 助手 |
| 📋 | 大纲视图 |

## 核心功能

### 文档编辑
- 触控优化的编辑器
- 双指缩放调整字号
- 长按选中文字
- 选中文本后弹出格式工具栏

### AI 起稿
移动端专用的起稿入口：
1. 点击底部 **起稿** 按钮
2. 输入文档标题和写作说明
3. AI 生成完整草稿
4. 预览后选择接受或放弃

### AI 对话
- 点击底部 **AI** 按钮打开对话面板
- 全屏对话界面
- 支持所有桌面端 AI 功能
- 滑动关闭

### 大纲导航
- 点击底部 **大纲** 按钮
- 底部弹出大纲面板
- 点击标题快速跳转

## 导出功能

在顶部 **更多菜单** 中：
- **复制 Markdown** - 复制到剪贴板
- **下载 .md** - 下载 Markdown 文件
- **导出 Word** - 生成 .docx 文件
- **导出 PDF** - 生成 PDF 文件

## 手势操作

| 手势 | 操作 |
|------|------|
| 单指点击 | 定位光标 |
| 长按 | 选中文字 |
| 双指捏合 | 缩放页面 |
| 左滑 | 返回上一页 |

## 保存状态

移动端保存状态显示在底部导航栏上方：
- **"保存中..."** - 正在同步
- **"已保存"** - 同步完成（2秒后自动消失）
- **"保存失败"** - 网络错误（3秒后消失）

## 离线使用

- 文档会缓存到本地
- 离线时显示云朵带斜杠图标
- 恢复网络后自动同步
