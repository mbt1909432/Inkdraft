# Documents List - 文档管理

![](../screenshot-documents.png)

文档列表页是管理所有文档的入口，支持文件夹组织、搜索筛选、批量操作等功能。

## 侧边栏功能

### 新建文档
点击 **+ 新建文档** 按钮创建新文档：
- **空白文档** - 从零开始
- **从模板创建** - 使用预设模板
- **导入 Markdown** - 上传 .md 文件

### 新建文件夹
点击文件夹图标创建新文件夹，用于分类管理文档。

### 文档列表
- 显示所有文档，按更新时间排序
- 置顶文档显示在顶部，带📌图标
- 点击文档名直接打开编辑

### 文件夹导航
- 点击文件夹筛选该文件夹下的文档
- 点击"全部文档"显示所有文档
- 支持嵌套文件夹结构

## 文档操作

每个文档右侧有 **⋯** 菜单，提供以下操作：

| 操作 | 说明 |
|------|------|
| **重命名** | 修改文档标题 |
| **置顶/取消置顶** | 将文档固定在列表顶部 |
| **删除** | 移动到回收站 |

### 批量操作
1. 点击 **选择** 按钮进入批量模式
2. 勾选多个文档
3. 点击 **删除选中** 批量删除

## 搜索与筛选

### 快速搜索
在搜索框输入关键词，实时过滤文档列表。

### 筛选选项
- **全部** - 显示所有文档
- **最近** - 按更新时间排序
- **已置顶** - 只显示置顶文档

## 导入功能

### 导入 Markdown 文件
1. 点击 **新建文档** → **导入 Markdown**
2. 选择本地 .md 或 .markdown 文件
3. 文件内容自动导入并创建新文档

### 代码块自动处理
导入时自动处理代码块：
- 无语言标识的代码块自动添加 `markdown` 标识
- 保留原有的代码高亮设置

## 响应式设计

### 桌面端
- 侧边栏常驻显示
- 支持拖拽调整宽度
- 文档列表与编辑器同屏显示

### 移动端
- 侧边栏以抽屉形式显示
- 点击菜单图标打开
- 点击文档自动跳转编辑
