# AI Assistant - AI 对话助手

![](../screenshot-editor-chat.png)

Inkdraft 内置了强大的 AI 对话助手，不仅能给出建议，还能**直接修改你的文档**。

## 核心能力：Agent 直接操作文档

传统编辑器中，AI 只能给出修改建议，你需要手动复制粘贴。Inkdraft 的 Agent 可以：

| 操作 | 能力 | 示例 |
|------|------|------|
| **增** | 补充内容 | "帮我补充一段关于 X 的说明" |
| **删** | 删除内容 | "删除所有重复的段落" |
| **改** | 修改内容 | "把这段话改得更专业一些" |
| **查** | 查询信息 | "总结这篇文档的主要观点" |

## 如何使用

### 1. 打开 AI 面板
- 点击工具栏的 **AI Assistant** 按钮
- 右侧会展开 AI 对话面板

### 2. 开始对话
直接在输入框中输入你的问题或指令，例如：
- "帮我润色这段文字"
- "给这篇文章写个摘要"
- "把这个列表转换成表格"

### 3. Agent 执行
Agent 会理解你的意图，并**直接在编辑器中执行修改**，你可以：
- 立即看到修改结果
- Ctrl+Z 撤销不满意的修改
- 继续对话进行进一步调整

## 选中文本操作

选中任意文本后，会自动弹出操作工具栏：

| 按钮 | 功能 |
|------|------|
| **B** | 加粗 |
| **I** | 斜体 |
| **U** | 下划线 |
| **S** | 删除线 |
| **代码** | 行内代码 |
| **润色** | AI 润色选中文字 |
| **扩写** | AI 扩展内容 |
| **缩写** | AI 精简内容 |
| **更多** | 更多 AI 操作选项 |

## 上下文记忆

AI Agent 会记住对话历史，支持多轮对话：
- "把刚才那段再改短一点"
- "换一种说法"
- "继续刚才的话题"

Token 使用量会实时显示，帮助你了解上下文消耗。

## Acontext SDK 代码执行

在 AI 对话中，Agent 可以运行 Python 代码：

```
用户: 帮我分析一下销售数据

Agent: 好的，我来运行分析代码...
[执行 Python 代码]
分析结果如下：
- 总销售额: ¥1,234,567
- 最高地区: 华东 (35%)
...
```

分析结果可以直接插入到文档中。
